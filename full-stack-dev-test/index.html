challenge : Code a RESTFUL service

It can be coded in the language of your choice. It must contain the following:

  • A minimum of 1 POST endpoint with a body
  • A minimum of 1 GET endpoint with a response
  • A simple UI to interact with both endpoints
  • An in-memory service database with 1 lookup query and 1 save query

What we will be interested in

  • How you structure your code (SOLID principles matter a lot to us!).
  • Validation
  • Reusability
  • The quality of the code you write
  • How you use the GitHub Flow

To submit you code

  • Create a GitRub repository to host your code
  • Add @nestordaza-dteck as collaborator
  • Create a Pull Request containing the code you wrote and assign him as reviewer

Good luck!
